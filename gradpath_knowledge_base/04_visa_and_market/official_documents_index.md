# Official Government Source Documents — Index (2026)

> Index of authoritative primary-source documents downloaded directly from
> Irish government websites. These documents are the most-current authoritative
> reference for visa and employment-permit information.

## ⭐ Most important document

`official_mar_roadmap_review_2025.pdf` is the authoritative source for the
**post-1-March-2026 salary thresholds**. Any threshold figures referenced
elsewhere in this knowledge base should be cross-checked against this PDF.

## All documents in this folder

- **Source authority**: Department of Enterprise, Tourism and Employment
- **Original URL**: https://enterprise.gov.ie/en/publications/publication-files/employment-permits-minimum-annual-remuneration-outcome-of-the-roadmap-review-2025.pdf
- **Criticality**: HIGHEST — supersedes any older threshold information
- **Description**: Authoritative source for 2026 salary thresholds and the gradual increase roadmap through 2030

## How the assistant should use these documents

The assistant treats these PDFs as **authoritative primary sources**. When the
user asks about visa thresholds, eligibility rules, or application procedures:

1. **For 2026 threshold figures**, retrieve from
   `official_mar_roadmap_review_2025.pdf` first.
2. **For procedural / application questions**, retrieve from
   `official_employment_permits_faqs_2024.pdf` and
   `official_critical_skills_permit_checklist.pdf`.
3. **For Stamp 1G / graduate-programme questions**, retrieve from
   `official_third_level_graduate_programme.pdf`.
4. **For Critical Skills List composition**, retrieve from
   `official_occupations_lists_review_dec_2023.pdf` (note: list updates may
   have happened after this review — flag this caveat to the user).

The hand-written summary files (`visa_*.md`) provide accessible explanations
in the assistant's voice, but the primary-source PDFs provide the actual
authoritative text. When a question is precise (e.g. "what is the exact
threshold for X type of permit"), the assistant should defer to the PDFs.
